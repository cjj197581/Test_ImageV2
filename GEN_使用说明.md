# gen.py 批量文生图工具使用说明

## 一句话概述

将指定文件夹中的所有 `.txt` prompt 文件，批量生成同名的 `.png` 图片。

## 快速开始

```cmd
python gen.py "你的prompts文件夹"
```

或双击 `run_gen2.bat`（已预设好路径）。

## 示例

```cmd
# 基础用法：生成文件夹中所有 .txt 对应的图片
python gen.py "需要生成的图片的提示词"

# 正方形 1:1
python gen.py "prompts" --size 1024x1024

# 输出到指定目录
python gen.py "prompts" --out ./slides

# 快速预览（低画质 + VIP 模型）
python gen.py "prompts" --model gpt-image-2-vip --quality low

# 横版 16:9 + 详细日志
python gen.py "prompts" --size 1792x1024 --verbose
```

## 命令行参数

| 参数 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `folder` | **是** | — | 包含 `.txt` prompt 文件的文件夹路径 |
| `--size` | 否 | `1792x1024` | 输出尺寸，如 `1024x1024` `1792x1024` `1280x720` |
| `--out` | 否 | 与 `.txt` 同目录 | 图片输出文件夹 |
| `--model` | 否 | `gpt-image-2` | 模型：`gpt-image-2` / `gpt-image-2-vip` / `gpt-image-2-all` |
| `--quality` | 否 | `high` | 画质（仅官方模型）：`high` / `medium` / `low` |
| `--verbose` / `-v` | 否 | 关 | 显示详细保存路径 |

## 模型选择指南

| 模型 | 中文文字 | 尺寸控制 | 速度 | 价格 | 适用场景 |
|------|----------|----------|------|------|----------|
| `gpt-image-2` | ★★★★★ | 任意尺寸 | 慢 | ~$0.21 | 教学PPT、海报、中文密集 |
| `gpt-image-2-vip` | ★★★★ | 30档预设 | 中 | $0.03 | 常规配图、社媒 |
| `gpt-image-2-all` | ★★★ | 不支持size | 快 | $0.03 | 简单配图、原型 |

## 画质选择指南

| 画质 | 速度 | 价格 | 适用场景 |
|------|------|------|----------|
| `high` | 130-240s | ~$0.21 | 最终成品、PPT、印刷 |
| `medium` | 25-40s | ~$0.05 | 审阅稿、迭代 |
| `low` | 3-8s | ~$0.006 | 快速预览、原型验证 |

## 常用尺寸

| 尺寸 | 比例 | 适用场景 |
|------|------|----------|
| `1792x1024` | 16:9 | PPT横版、视频封面 |
| `1024x1792` | 9:16 | 手机海报、竖版PPT |
| `1024x1024` | 1:1 | 社交媒体、头像 |
| `1280x720` | 16:9 | 网页banner |
| `2048x1152` | 16:9 | 2K横版PPT |

## 行为说明

- **同名覆盖**：`slide_01.txt` → `slide_01.png`，如果已存在则**跳过不重复生成**
- **空文件跳过**：内容为空的 `.txt` 自动跳过
- **非 `.txt` 忽略**：只处理 `.txt` 文件，其他格式无视
- **错误不中断**：某张失败不影响后续生成，最后统一汇总

## run_gen2.bat 批命令

```batch
@chcp 65001 >nul
python gen.py "需要生成的图片的提示词"
PAUSE
```

- `@chcp 65001 >nul` — 将 CMD 切换到 UTF-8，解决中文路径和输出乱码
- `.bat` 文件必须用 **GBK/ANSI 编码**保存（Windows 记事本默认）
- 如需指定其他文件夹，修改第二行的路径即可

## 典型工作流

```cmd
# 1. 先用低画质快速验证 prompt 效果（几秒一张）
python gen.py "prompts" --quality low --size 1024x1024

# 2. 确认满意后，高画质出最终稿
python gen.py "prompts" --quality high --size 1792x1024
```

## 注意事项

- 确保 `.env` 中 `API_KEY` 有效且账户余额充足
- high 画质每张约 $0.21，生成 5 张约 $1
- 复杂中文 prompt 建议用 `gpt-image-2` 官方模型
- 如遇 403 错误，检查 API易 账户余额
